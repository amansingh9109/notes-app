# notes-app
Full-stack Notes App with Spring Boot, MySQL,and modern UI created by Aman Singh and Priyanshu Mourya.

Features
- Add notes  
- Edit notes  
- Delete notes  
- Search notes  
- Upload image with notes  
- Preview image before saving  
- Read more / show less for long notes  
- Supports paragraphs (multi-line text)  

Concepts Used
- Object-Oriented Programming (OOP)  
- REST API development  
- CRUD operations (Create, Read, Update, Delete)  
- Client-Server architecture  
- Database integration (MySQL)  
- Frontend and backend integration

How to Run

1. Clone the repository:
   git clone https://github.com/amansingh9109/notes-app.git
2. Open the project in IntelliJ IDEA
3. Make sure MySQL is running
4. Update database details in `application.properties`
5. Run the Spring Boot application
6. Open your browser and go to:
   http://localhost:8080
